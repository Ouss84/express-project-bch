3 Create data access layer for the data

Create a datastorage layer with all necessary methods.

Necessary operations:

Your data access layer should have the following operations:
  • get all
  • get one
  • insert
  • update
  • remove
The data is read dynamically from the data file. When the data is changed it is saved back to the disk file.

4 Create server (index.js)

Create a server. Server is listening host localhost:3000. Main menu can be opened from root route /. Menu should include links to other pages.

5 Create webpages

Create webpages for the following operations:
  • get all
  • get one
  • insert
  • update
  • remove

Below (Appendix) are a few examples of web pages. Design and implement the main menu and all other necessary pages.

Design and implement stylesheets for your application.

6 Submit the project

Submit your project folder Bahri_Oussama_computer_project acording to instructions given separately. Remove the node_modules folder before submitting. The project folder should include package.json file
The project folder should also include the data in a json file named as Bahri_Oussama_computers.json.
